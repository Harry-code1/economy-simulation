# Economy Simulation

A simple text-based economy and resource management simulation written in Python.

This project simulates an economy with money, population, power, factories, and power plants.  
Each turn (tick), the player makes decisions that affect future growth and stability.

---

## Features

- Tick-based simulation
- Player decisions (build factories or power plants)
- Resource management (money, population, power)
- Power shortages with penalties
- Stats screen for each tick
- Help / rules screen

---

## How to Run

1. Make sure you have Python 3 installed  
2. Download or clone the repository  
3. Run the simulation using:

```bash
python simulation.py
